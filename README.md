# Alfík · Detský portál

Interaktívna detská aplikácia — nasadenie na GitHub Pages.

## Spustenie lokálne

Otvor `index.html` v prehliadači (cez lokálny server, napr. `npx serve .`).

## GitHub Pages

1. Nahraj obsah tohto priečinka do GitHub repozitára
2. Nastav **Settings → Pages → Source: Deploy from branch → main / (root)**
3. Aplikácia bude dostupná na `https://<username>.github.io/<repo-name>/`

## Navigácia

| Screen | Popis |
|---|---|
| 2A · Úvod | Úvodná obrazovka s pozdravom a odznakmi |
| 2B · Mapa cesty | Herná mapa s zastávkami |
| 2C1 · Ihrisko | Výber kategórií |
